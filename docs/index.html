Qwen3 TTS 1.7B

Qwen3 自定义语音合成

基于 Qwen3 TTS 1.7B 模型的 OpenAI 兼容语音合成 API 服务。支持自定义音色、精细化参数控制，Docker 一键部署，为您的应用赋予自然流畅的语音表达能力。

快速开始 API 文档
功能特性

为语音合成而生的服务端方案

从模型加载到音频输出，每一步都经过精心设计，确保稳定、高效、易于集成。

OpenAI 兼容接口

完全兼容 /v1/audio/speech 端点，可直接替换 OpenAI TTS，无需修改现有代码。

异步非阻塞推理

推理任务在线程池中执行，异步事件循环保持响应，健康检查永不阻塞。

Docker 多阶段构建

编译阶段与运行阶段分离，最终镜像精简高效，内置健康检查与 GPU 支持。

精细化参数控制

支持 temperature、top_p、top_k、repetition_penalty 等参数，并可独立调节 Subtalker 子模型行为。

WAV / MP3 双格式

通过 response_format 参数在 WAV 和 MP3 之间自由切换，满足不同场景需求。

超时保护

可配置推理超时和最大 Token 数，防止异常请求占用 GPU 资源，保障服务稳定性。

快速开始

Docker 一键部署

只需准备模型文件，一条命令即可启动完整的 TTS 服务。

shell 设置模型路径并启动 
# 指定模型路径（如使用非默认路径）
export QWEN3_TTS_MODEL_PATH=/path/to/qwen3-tts-1.7b-customvoice

# 构建并启动
docker compose up -d

# 验证服务状态
curl http://localhost:8001/health
API 参考

接口端点

提供两个语音合成端点和一个健康检查端点，兼容 OpenAI SDK。

GET /health 健康检查，始终即时响应
POST /v1/audio/speech OpenAI 兼容语音合成
POST /v1/audio/tts 简化接口，支持服务端保存

POST /v1/audio/speech 请求示例

POST /v1/audio/speech curl 
curl -X POST http://localhost:8001/v1/audio/speech \
  -H "Content-Type: application/json" \
  -d '{
    "input": "你好，欢迎使用 Qwen3 语音合成服务。",
    "voice": "Vivian",
    "response_format": "mp3"
  }' \
  --output speech.mp3
POST /v1/audio/speech Python SDK 
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8001/v1",
    api_key="not-needed"
)

with client.audio.speech.with_streaming_response.create(
    model="qwen3-tts-1.7b-customvoice",
    voice="Vivian",
    input="你好，这是一段测试语音。",
) as response:
    response.stream_to_file("output.wav")

生成参数一览

input string 需要合成语音的文本内容
voice string 说话人名称，默认 "Vivian"
response_format string 音频格式，支持 "wav" 或 "mp3"
temperature float 采样温度，默认 0.2
top_p float Nucleus 采样阈值，默认 0.6
top_k int Top-K 采样，默认 15
repetition_penalty float 重复惩罚系数，默认 1.1
seed int 随机种子，不传则自动生成
instruct string 模型指令提示，可选
subtalker_temperature float Subtalker 子模型温度，可选
subtalker_top_p float Subtalker 子模型 top_p，可选
subtalker_top_k int Subtalker 子模型 top_k，可选
配置参考

环境变量与启动参数

所有配置项均可通过环境变量或命令行参数设定，灵活适配不同部署环境。

环境变量

变量名默认值说明
QWEN3_TTS_MODEL_PATH/data/models/qwen3-tts-1.7b-customvoice模型权重路径
QWEN3_TTS_DEVICEcuda:0PyTorch 推理设备
QWEN3_TTS_PORT8001对外暴露端口
QWEN3_TTS_TIMEOUT120单次推理超时秒数
QWEN3_TTS_MAX_TOKENS3072单次生成最大 Token 数

命令行参数

shell
python qwen3-tts-server.py \
  --host 0.0.0.0 \
  --port 8001 \
  --device cuda:0 \
  --workers 1
参数默认值说明
--host0.0.0.0绑定地址
--port8001监听端口
--devicecuda:0推理设备
--workers1Uvicorn worker 数量（推理使用独立线程池）
Docker

镜像构建与运行

多阶段构建将编译工具链与运行时分离，最终镜像体积精简。

shell 独立构建（不使用 Compose） 
docker build -t qwen3-tts:local .
docker run --gpus all \
  -v /data/models:/data/models:ro \
  -p 8001:8001 \
  qwen3-tts:local
